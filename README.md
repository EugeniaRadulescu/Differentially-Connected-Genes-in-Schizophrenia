# Differentially Connected Genes in Schizophrenia

This repository contains the code for MS: Analysis of gene co-expression networks connectivity patterns implicates aberrant neuron-oligodendroglia interactions in schizophrenia
Bulk RNA-Seq, snRNA-Seq and organoids expression data are available on Zenodo: https://doi.org/10.5281/zenodo.19673070
